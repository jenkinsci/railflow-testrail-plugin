(optional) Test Run name in TestRail.
If not specified, Railflow will automatically create the Test Run with format: {jenkins_job}-{build_number}
If the Test Run does not exist, Railflow will automatically create the Test Run with the specified name.
If the Test Run exists, Railflow will add the test results in the specified Test Run.
If "Test Run ID" is set, the value of this property will be ignored.
